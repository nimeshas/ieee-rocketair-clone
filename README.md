# ieee-rocketair-clone
frontend task for ieee-replicating any one page with navbar and footer

Project Setup:
I have deployed this website using Vercel, Link to access the website: https://ieee-rocketair-clone.vercel.app/
Alternatively, it can be accessed through my github repository: https://github.com/nimeshas/ieee-rocketair-clone.git

Features:
1)Navbar:
A logo positioned on the left.
A "Get in touch" button with redirects to the contact page.
A burger menu on the right using a four-dot design.

2)Partner page:
A horizontally scrolling partner logo section using `Splide.js`.
Each slide contains the company logo and a small description.
A draggable custom cursor effect appears when interacting with elements.

3)Footer:
The page is split into two halves
Left section:
    A heading, an email input field and a 'call to action' subscribe button
Right section:
  A vertical menu with links.
  Social media icons with hover effects, that redirect to the social media pages and privacy policy link.

Implementation:

1) I started by using basic HTML and CSS to create a general structure of the navbar and the footer
2) then I inspected the page to figure out some of the more detailed aspects of the page and more accurate measurements.
3) I also used Bootstrap to help make the page more responsive and the styling a little easier.
4) Following this I tried to replicate some of the animations, using CSS only.
5) For the footer, I used FontAwesome for the social media icons.
6) I then cloned the partner's page which has a heading, followed by a carousel.
7) I followed the same procedure as I did for the navbar and footer.
8) Once I created the basic structure and styling, I used Splide.js for the horizontal scrolling section.

Since this was one of my first web development projects, I was not able to implement all the smooth animations very well, If given a little bit more time, I would ensure that the site is more responsive throughout different screen sizes even though i tried my best to maintain responsiveness.
